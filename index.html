爆弾職人Polka

作品概要

全機種対応の2Dアクションゲームです。
プレイヤーは爆弾を使ってマップ内の敵をすべて倒すことでゲームクリアを目指します。
戦略的な爆弾配置が鍵となります。

担当箇所

Input System を用いた全機種対応の入力処理を実装し、CSVデータによるステージ描画システムを構築しました。爆弾は通常・天井（上）破壊用・床（下）破壊用・地雷の4種類を実装し、マス単位での管理ロジックにより爆発範囲や地形破壊を制御しています。また、ジャンプ入力には先行入力を導入し、天井の有無に応じてジャンプ力を切り替えることで、状況に応じた上下移動を可能にしました。そのほか、鍵によるステージ解放制御、視野範囲を動的に変更できるシステム、シーン管理および各種マネージャーの実装、タイトル画面の設定、爆弾・床爆発のアニメーション制御、制限時間管理、UI全般の実装、全体の機能結合までを担当しました。

Trick or Find

作品概要

筐体対応の2DアクションRPGです。
学園祭に展示するために作成したゲームです。
プレイヤーは銃を使い、マップ内の敵を倒しながら宝を収集してスコアを獲得します。 状況に応じた戦略的なモード切替が高スコア獲得の鍵となるゲームです。

担当箇所

ゲーム全体を制御するシーン管理およびマネージャー機能の設計・実装を担当しました。基底クラスを用いた設計により、3種類の宝ごとの挙動を継承・分岐処理として実装し、拡張性を意識した構成としています。また、タイトル画面を含むフェードイン／フェードアウト演出、プレイヤーおよび敵キャラクターのアニメーション制御、サーチモードの探索ロジック、スコア管理、UI全般の実装、各機能の結合、筐体向け操作説明表示までを一貫して担当しました。


詳細説明

本作は、“宝探し” と “バンサバ（弾幕・サバイバル系アクション）” を融合させた新感覚のアクションゲームです。プレイヤーは 「銃モード」 と 「サーチモード」 の2つを切り替えながら、敵の群れを突破しつつ宝の在りかを探り当てることを目指します。銃モード では射撃が可能で、迫り来る敵を撃退しながら安全圏を確保できます。一方、サーチモード では攻撃ができない代わりに、宝の位置が「良い・悪い」といった形で徐々に絞り込まれ、探索において重要な手がかりを得られます。また、サーチモード時には敵が波のような特殊パターンで襲来し、通常とは異なる緊張感が生まれます。さらに、プレイヤーの周囲の視界は円状に徐々に狭まり、画面外側が黒く覆われていく独自のギミックが特徴です。この視界は一定時間が経過すると回復し、リズミカルな探索テンポを生み出します。銃撃と探索、緊張と回復が入り混じる、メリハリのあるアクション体験を楽しめる作品となっています。
